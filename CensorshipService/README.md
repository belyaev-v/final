# Censorship Service

Сервис для валидации комментариев на наличие запрещенных слов.

## Запуск

```bash
go run main.go
```

Или с параметрами:

```bash
go run main.go -port=8083
```

## Эндпоинты

- `POST /validate` - валидация текста комментария
  - Body: `{"text": "Текст комментария"}`
  - Ответ: `200 OK` если допустим, `400 Bad Request` если содержит запрещенные слова

## Запрещенные слова

- qwerty
- йцукен
- zxvbnm

## Особенности

- Middleware для request_id и логирования
- Проверка без учета регистра
